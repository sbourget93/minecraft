DrZhark's Custom Mob Spawner v3.3.0
(c) DrZhark, Bloodshot 2011 - 2014
version 3.3.0 for minecraft 1.7.10

To install:
copy the CustomMobSpawner.zip file into /.minecraft/mods

Documentation:
CustomMobSpawner UserGuide can be found in root of zip file.

For modders:
You need to create a package inside the 'common' package named 'drzhark.customspawner' and paste the CustomSpawner.java there.  Then you can add: import drzhark.customspawner.CustomSpawner; in your mod's main file.

When distribuiting your mod, please remove the customspawner file from it, please ask the users to download the file from the website.

An example of how to set and use CustomSpawner will be available on the forums.
